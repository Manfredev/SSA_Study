A. Prerequisite Mathematics

Why This Matters

Every derivation on the SSA exam—from PDFs and CDFs to variance reduction—relies on calculus skills. Rusty integration will slow you down under exam pressure. This section sharpens the specific techniques you need.

A.1 Integration by Parts

Key Formula FS: Integration

$$\int u\,dv = uv - \int v\,du$$

Choose $u$ and $dv$ using the LIATE rule (priority order):

  1. Logarithmic — $\ln x$, $\log x$
  2. Inverse trigonometric — $\arctan x$, $\arcsin x$
  3. Algebraic — $x^n$, polynomials
  4. Trigonometric — $\sin x$, $\cos x$
  5. Exponential — $e^x$, $a^x$

Set $u$ = whichever factor appears earliest in LIATE, and $dv$ = everything else $\cdot\,dx$.

Worked Example A.1: $\int x e^x\,dx$

Step 1. Identify parts using LIATE. $x$ is Algebraic (A), $e^x$ is Exponential (E). Since A comes before E, let $u = x$.

$u = x \implies du = dx$

$dv = e^x\,dx \implies v = e^x$

Step 2. Apply the formula:

$$\int x e^x\,dx = xe^x - \int e^x\,dx = xe^x - e^x + C$$

Result: $\int x e^x\,dx = xe^x - e^x + C = e^x(x - 1) + C$

Worked Example A.2: $\int x^2 e^{-x}\,dx$

Step 1. $x^2$ is Algebraic, $e^{-x}$ is Exponential. Let $u = x^2$, $dv = e^{-x}\,dx$.

$u = x^2 \implies du = 2x\,dx$,   $v = -e^{-x}$

Step 2. First application:

$$\int x^2 e^{-x}\,dx = -x^2 e^{-x} + \int 2x e^{-x}\,dx$$

Step 3. Apply integration by parts again to $\int 2x e^{-x}\,dx$. Let $u = 2x$, $dv = e^{-x}\,dx$:

$u = 2x \implies du = 2\,dx$,   $v = -e^{-x}$

$$\int 2x e^{-x}\,dx = -2xe^{-x} + \int 2e^{-x}\,dx = -2xe^{-x} - 2e^{-x}$$

Step 4. Combine:

$$\int x^2 e^{-x}\,dx = -x^2 e^{-x} - 2xe^{-x} - 2e^{-x} + C = -e^{-x}(x^2 + 2x + 2) + C$$

Worked Example A.3: $\int \ln(x)\,dx$

Step 1. $\ln(x)$ is Logarithmic, the implicit $1$ is Algebraic. Let $u = \ln x$, $dv = dx$.

$u = \ln x \implies du = \frac{1}{x}\,dx$,   $v = x$

Step 2. Apply:

$$\int \ln(x)\,dx = x\ln x - \int x \cdot \frac{1}{x}\,dx = x\ln x - \int 1\,dx = x\ln x - x + C$$

A.2 Exponential Integrals

Key Patterns

$$\int e^{a - x}\,dx = -e^{a-x} + C$$

More generally: $\int e^{kx}\,dx = \frac{1}{k}e^{kx} + C$ for any constant $k \neq 0$.

For $e^{a-x}$ we have $k = -1$, so the antiderivative picks up a factor of $-1$.

Worked Example A.4: Show $\int_1^3 \frac{e^{3-x}}{e^2 - 1}\,dx = 1$

Step 1. Factor out the constant:

$$\int_1^3 \frac{e^{3-x}}{e^2 - 1}\,dx = \frac{1}{e^2 - 1}\int_1^3 e^{3-x}\,dx$$

Step 2. Evaluate the integral:

$$\int_1^3 e^{3-x}\,dx = \left[-e^{3-x}\right]_1^3 = \left(-e^{3-3}\right) - \left(-e^{3-1}\right) = -e^0 + e^2 = -1 + e^2$$

Step 3. Combine:

$$\frac{1}{e^2 - 1}\cdot(e^2 - 1) = 1 \quad \checkmark$$

Result: The integral equals exactly 1, confirming this is a valid (sub-)density on $[1,3]$.

A.3 Arctan: Derivative and Integral

Key Formulas

$$\frac{d}{dx}\arctan(x) = \frac{1}{1 + x^2}$$

$$\int \frac{1}{1 + x^2}\,dx = \arctan(x) + C$$

Shifted form: $\int \frac{1}{1 + (x - a)^2}\,dx = \arctan(x - a) + C$

Background: Why the shifted form works
Substitution for Shifted Arguments

When you see $\frac{1}{1 + (x - a)^2}$, let $t = x - a$ so $dt = dx$. Then:

$$\int \frac{1}{1 + (x-a)^2}\,dx = \int \frac{1}{1 + t^2}\,dt = \arctan(t) + C = \arctan(x - a) + C$$

No chain rule factor appears because $\frac{d}{dx}(x - a) = 1$. If it were $(2x - a)$, you would get a factor of $\frac{1}{2}$.
Worked Example A.5: $\int_3^4 \frac{1}{1 + (x - 3)^2}\,dx$

Step 1. Use the shifted arctan formula directly:

$$\int_3^4 \frac{1}{1 + (x-3)^2}\,dx = \left[\arctan(x - 3)\right]_3^4$$

Step 2. Evaluate at the bounds:

$$= \arctan(4 - 3) - \arctan(3 - 3) = \arctan(1) - \arctan(0) = \frac{\pi}{4} - 0 = \frac{\pi}{4}$$

Result: $\displaystyle\int_3^4 \frac{1}{1 + (x-3)^2}\,dx = \frac{\pi}{4}$

Common Mistakes

  • Forgetting $\arctan(0) = 0$ and $\arctan(1) = \pi/4$ (not $\pi/2$!).
  • Confusing $\arctan(\infty) = \pi/2$ with $\arctan(1) = \pi/4$.
  • Missing the minus sign on $\int e^{a-x}\,dx = -e^{a-x}$ (the exponent has a hidden $-1$ coefficient on $x$).

B. PDF / CDF / Inverse Transform

Why This Matters

Generating random variates from arbitrary distributions is the backbone of simulation. You must be able to verify a PDF, derive the CDF, invert it, and apply the composition method for piecewise densities.

B.1 Verifying a Valid PDF

Two Conditions FS: Distributions

  1. Non-negativity: $f(x) \geq 0$ for all $x$.
  2. Normalization: $\displaystyle\int_{-\infty}^{\infty} f(x)\,dx = 1$.

For piecewise PDFs, check each piece is non-negative and that the pieces sum to 1.

B.2 CDF Derivation from a PDF

The cumulative distribution function (CDF) is:

$$F(x) = \int_{-\infty}^{x} f(t)\,dt$$

For a piecewise PDF defined on $[a, b]$, the CDF is found piece by piece, ensuring continuity at each boundary: the CDF from the left piece must equal the CDF from the right piece at each breakpoint.

B.3 Inverse Transform Method

Inverse Transform Method — Step by Step

Derive the CDF $F(x)$
Integrate the PDF: $F(x) = \int_{-\infty}^x f(t)\,dt$. Make sure $F$ is continuous and goes from 0 to 1.
Set $F(x) = U$ where $U \sim \text{Uniform}(0,1)$
We want to solve for $x$ in terms of $U$.
Solve for $x = F^{-1}(U)$
Algebraically invert the CDF. This gives your variate generation formula.
$X = F^{-1}(U)$ has distribution $F$
Generate: draw $U$, compute $X = F^{-1}(U)$
Each uniform random number produces one random variate from the target distribution.
Tip: If $F^{-1}$ has no closed form, use the composition method instead.

B.4 Composition Method

Composition Decomposition FS: Composition

If $f(x) = p_1 f_1(x) + p_2 f_2(x) + \cdots + p_k f_k(x)$ where $p_i > 0$, $\sum p_i = 1$, and each $f_i$ is a valid PDF:

  1. Generate $U_1 \sim \text{Uniform}(0,1)$.
  2. Select component $i$ with probability $p_i$ (using $U_1$).
  3. Generate $X$ from $f_i$ (using one or more additional uniforms).

The weights $p_i$ equal the integrals of each piece of the original PDF, and each $f_i$ is the normalized sub-density (the piece divided by its weight).

B.5 Simple Example: Power PDF

Worked Example B.1: $f(x) = \tfrac{3}{8}x^2$ on $[0, 2]$

Step 1. Verify it is a valid PDF:

$f(x) = \frac{3}{8}x^2 \geq 0$ on $[0,2]$. Check normalization:

$$\int_0^2 \frac{3}{8}x^2\,dx = \frac{3}{8}\left[\frac{x^3}{3}\right]_0^2 = \frac{3}{8}\cdot\frac{8}{3} = 1 \quad\checkmark$$

Step 2. Derive the CDF:

$$F(x) = \int_0^x \frac{3}{8}t^2\,dt = \frac{3}{8}\cdot\frac{x^3}{3} = \frac{x^3}{8}, \quad 0 \leq x \leq 2$$

Step 3. Invert: Set $U = x^3/8$, solve for $x$:

$$x^3 = 8U \implies x = 2U^{1/3}$$

Generator: $X = 2U^{1/3}$ where $U \sim \text{Uniform}(0,1)$.

B.6 Key Exam PDF — Piecewise with Composition

The Exam PDF

$$f(x) = \begin{cases} \dfrac{1}{2(e^2-1)}e^{3-x} & 1 \leq x \leq 3 \\[8pt] \dfrac{2}{\pi(1+(x-3)^2)} & 3 < x \leq 4 \\[8pt] 0 & \text{otherwise} \end{cases}$$

Worked Example B.2: Full derivation — verification, CDF, inverse, composition

Step 1: Verify the PDF (show total integral = 1)

Piece 1 on $[1,3]$:

$$\int_1^3 \frac{1}{2(e^2-1)}e^{3-x}\,dx = \frac{1}{2(e^2-1)}\left[-e^{3-x}\right]_1^3 = \frac{1}{2(e^2-1)}\left(-e^0 + e^2\right) = \frac{e^2 - 1}{2(e^2-1)} = \frac{1}{2}$$

Piece 2 on $(3,4]$:

$$\int_3^4 \frac{2}{\pi(1+(x-3)^2)}\,dx = \frac{2}{\pi}\left[\arctan(x-3)\right]_3^4 = \frac{2}{\pi}\left(\arctan(1) - \arctan(0)\right) = \frac{2}{\pi}\cdot\frac{\pi}{4} = \frac{1}{2}$$

Total: $\frac{1}{2} + \frac{1}{2} = 1$   $\checkmark$

Step 2: Identify the composition weights and sub-densities

Since each piece integrates to $\frac{1}{2}$, we have $p_1 = \frac{1}{2}$ and $p_2 = \frac{1}{2}$.

The normalized sub-densities are obtained by dividing each piece by its weight:

  • Sub-density 1: $f_1(x) = \dfrac{f(x)}{p_1} = \dfrac{e^{3-x}}{e^2-1}$   on $[1,3]$
  • Sub-density 2: $f_2(x) = \dfrac{f(x)}{p_2} = \dfrac{4}{\pi(1+(x-3)^2)}$   on $(3,4]$

Step 3: CDF and inverse for sub-density 1

$$F_1(x) = \int_1^x \frac{e^{3-t}}{e^2-1}\,dt = \frac{1}{e^2-1}\left[-e^{3-t}\right]_1^x = \frac{1}{e^2-1}\left(-e^{3-x}+e^2\right) = \frac{e^2 - e^{3-x}}{e^2 - 1}$$

Set $F_1(x) = U$:

$$U = \frac{e^2 - e^{3-x}}{e^2 - 1} \implies e^{3-x} = e^2 - U(e^2 - 1)$$

$$3 - x = \ln\!\left(e^2 - U(e^2-1)\right) \implies x = 3 - \ln\!\left(e^2 - U(e^2-1)\right)$$

$F_1^{-1}(U) = 3 - \ln\!\left(e^2 - U(e^2 - 1)\right)$

Step 4: CDF and inverse for sub-density 2

$$F_2(x) = \int_3^x \frac{4}{\pi(1+(t-3)^2)}\,dt = \frac{4}{\pi}\left[\arctan(t-3)\right]_3^x = \frac{4}{\pi}\arctan(x-3)$$

Set $F_2(x) = U$:

$$U = \frac{4}{\pi}\arctan(x-3) \implies \arctan(x-3) = \frac{\pi U}{4} \implies x = 3 + \tan\!\left(\frac{\pi U}{4}\right)$$

$F_2^{-1}(U) = 3 + \tan\!\left(\dfrac{\pi U}{4}\right)$

Step 5: Full Composition Algorithm

Composition Algorithm for the Exam PDF

Generate $U_1 \sim \text{Uniform}(0,1)$
This selects which component to use.
If $U_1 < 0.5$: use component 1 (exponential piece)
Generate $U_2 \sim \text{Uniform}(0,1)$. Return $X = 3 - \ln(e^2 - U_2(e^2 - 1))$.
$X \in [1, 3]$ from the exponential sub-density
Else ($U_1 \geq 0.5$): use component 2 (Cauchy-like piece)
Generate $U_2 \sim \text{Uniform}(0,1)$. Return $X = 3 + \tan(\pi U_2 / 4)$.
$X \in (3, 4]$ from the arctan sub-density

Common Mistakes with Composition

  • Forgetting to normalize sub-densities: If piece $i$ integrates to $p_i$, the sub-density is $f_i(x) = f(x)/p_i$, NOT $f(x)$ itself.
  • Wrong integration bounds: Each sub-density is only defined on its own interval. The CDF integration starts at the left boundary of that interval.
  • Confusing the two uniform draws: $U_1$ selects the component, $U_2$ generates the variate. They must be independent.

C. Poisson Processes

Why This Matters

Poisson processes model arrival patterns in queues, networks, and service systems. The exam tests merging, splitting, memorylessness, and utilization — all in applied scenarios.

C.1 Definition & Properties

Three Defining Properties FS: Poisson Process

  1. Independent increments: Numbers of arrivals in non-overlapping intervals are independent.
  2. Stationary increments: The distribution of arrivals depends only on interval length, not position.
  3. No simultaneous events: $P[\text{2 or more arrivals in } (t, t+h)] = o(h)$ as $h \to 0$.

Key Formulas

PMF: $\displaystyle P[N(t) = k] = \frac{(\lambda t)^k e^{-\lambda t}}{k!}$,   $k = 0, 1, 2, \ldots$

Interarrival times: $T_i \sim \text{Exp}(\lambda)$, i.e., $p(t) = \lambda e^{-\lambda t}$ for $t \geq 0$.

Mean interarrival time: $E[T] = 1/\lambda$.

C.2 Memoryless Property

Memoryless Property of the Exponential

$$P(T > s + t \mid T > s) = P(T > t)$$

If a server has been busy for 5 minutes (exponential service time), the remaining time has the same distribution as the original — the past is forgotten.

C.3 Merging Poisson Processes

Superposition

If independent Poisson processes with rates $\lambda_1, \lambda_2, \ldots, \lambda_k$ are merged, the result is a Poisson process with rate:

$$\lambda = \lambda_1 + \lambda_2 + \cdots + \lambda_k$$

C.4 Splitting / Thinning

Thinning

A Poisson process with rate $\lambda$ is split into types. Each arrival is independently classified as type $i$ with probability $p_i$. Then type $i$ arrivals form a Poisson process with rate $\lambda_i = p_i \lambda$.

If arrivals come from merging streams with rates $\lambda_i$, the probability a random arrival is type $i$ is $\lambda_i / \lambda$.

C.5 Competing Exponentials

Which Finishes First?

If $X \sim \text{Exp}(\lambda_X)$ and $Y \sim \text{Exp}(\lambda_Y)$ are independent:

$$P(X < Y) = \frac{\lambda_X}{\lambda_X + \lambda_Y}$$

$$\min(X, Y) \sim \text{Exp}(\lambda_X + \lambda_Y)$$

C.6 Utilization Factor

Server Utilization FS: Queuing

$$\rho = \lambda \cdot E[S]$$

where $\lambda$ = arrival rate and $E[S]$ = mean service time. For stability, $\rho < 1$.

C.7 Worked Example: Fitness Center Smoothie Bar

Worked Example C.1: Smoothie bar with merging, utilization, and competing exponentials

Setup: Protein smoothie orders arrive at rate 6/hr, fruit smoothie orders at rate 4/hr. Preparation time is Exp(mean 3 min). After prep, each smoothie requires exactly 1 min of processing.

(a) Merged rate and utilization

Merged arrival rate: $\lambda = 6 + 4 = 10$ per hour.

Mean service time: $E[S] = 3 + 1 = 4$ min $= \frac{4}{60} = \frac{1}{15}$ hr.

Utilization:

$$\rho = \lambda \cdot E[S] = 10 \times \frac{1}{15} = \frac{2}{3} \approx 0.667$$

Result: $\rho = 2/3$. The server is busy about 67% of the time.

(b) Scenario with competing exponentials

Setup: A customer arrives at the smoothie bar and sees 2 customers in the queue ahead. Meanwhile, a drink station next door has 1 customer being served with service time $\sim \text{Exp}(\text{rate } 1/2 \text{ per min})$. Which server will free up for the customer first?

Step 1: Apply the memoryless property.
The person at the drink station has already been there for some time. Normally you'd think "they must be almost done." But because the exponential distribution is memoryless: $$P(T > s + t \mid T > s) = P(T > t)$$ Their remaining time is still $\text{Exp}(\text{rate } 1/2)$ — as if they just started. This is the key insight that makes the problem tractable.

Step 2: Identify the two competing exponentials.
We have two servers working simultaneously:

  • Drink station: rate $= 1/2$ per min (mean 2 min to finish)
  • Smoothie bar (current customer): rate $= 1/3$ per min (mean 3 min prep time, ignoring the 1-min deterministic part for simplicity)

Step 3: Use the competing exponentials formula.
For two independent exponentials, the probability that one finishes before the other is: $$P(\text{Exp}(\lambda_1) < \text{Exp}(\lambda_2)) = \frac{\lambda_1}{\lambda_1 + \lambda_2}$$ The one with the higher rate is more likely to finish first. Here: $$P(\text{drink first}) = \frac{\lambda_{\text{drink}}}{\lambda_{\text{drink}} + \lambda_{\text{smoothie}}} = \frac{1/2}{1/2 + 1/3}$$

Step 4: Simplify the fraction.
The denominator is $1/2 + 1/3$. Find a common denominator (6): $$\frac{1}{2} + \frac{1}{3} = \frac{3}{6} + \frac{2}{6} = \frac{5}{6}$$ So: $$P(\text{drink first}) = \frac{1/2}{5/6} = \frac{1}{2} \times \frac{6}{5} = \frac{6}{10} = \frac{3}{5} = 0.6$$

Result: There is a 60% chance the drink station finishes its current customer before the smoothie bar finishes its current customer. The drink station has the higher rate ($1/2 > 1/3$), so it's more likely to finish first.

D. Hypothesis Testing Reference

Why This Matters

Choosing the correct statistical test and interpreting results correctly is essential for validating simulation output. The exam frequently asks you to pick the right test, compute the statistic, and state conclusions.

D.1 Which Test to Use — Decision Flowchart

Test Selection Flowchart

Is the goal to test if data fits a distribution?
Yes → Use Chi-squared Goodness of Fit or KS test.
No → Go to step 2.
Are you comparing two populations / systems?
Yes → Go to step 3.
No → Use a one-sample t-test or CI.
Are observations paired (same random seeds, before/after)?
Yes → Paired t-test.
No → Go to step 4.
Can you assume normality?
Yes → Welch two-sample t-test (unequal variances).
No → Wilcoxon rank-sum test (non-parametric).

D.2 Chi-Squared Goodness of Fit

Procedure FS: Chi-Squared GoF

Test statistic: $\displaystyle\chi^2 = \sum_{i=1}^{k} \frac{(O_i - E_i)^2}{E_i}$

where $O_i$ = observed count, $E_i$ = expected count in bin $i$.

Degrees of freedom:

  • All parameters known (fully specified $H_0$): $\text{df} = k - 1$
  • $m$ parameters estimated from data: $\text{df} = k - m - 1$

Bin merging rule: Merge adjacent bins until every bin has $E_i \geq 5$. After merging, $k$ is the new number of bins.

Common Mistake: Degrees of Freedom

If you estimate $\mu$ and $\sigma^2$ from the data to fit a normal distribution, then $m = 2$, and $\text{df} = k - 3$ (not $k - 1$). Forgetting to subtract estimated parameters inflates df, making the test less conservative.

D.3 Kolmogorov-Smirnov (KS) Test

KS Test — Three Adjustment Cases FS: KS Test

The KS statistic is $D_n = \sup_x |F_n(x) - F_0(x)|$, where $F_n$ is the empirical CDF and $F_0$ is the hypothesized CDF.

Case 1: All parameters known. Compute the adjusted statistic:

$$D_n^* = \left(\sqrt{n} + 0.12 + \frac{0.11}{\sqrt{n}}\right)D_n$$

Compare $D_n^*$ to critical values from the standard KS table.

Case 2: Normal with $\bar{X}$ and $S^2$ estimated. Use the Lilliefors critical values (larger than Case 1 critical values, because estimation makes the test less sensitive).

Case 3: Exponential with $\bar{X}$ estimated. Use the separate exponential KS critical values from the formula sheet.

Common Mistake: Using Standard KS Table with Estimated Parameters

If you estimated parameters from the data, you cannot use the standard KS critical values. You must use the adjusted tables (Lilliefors for normal, special table for exponential). Using the wrong table makes the test too liberal (rejects $H_0$ less often than it should).

D.4 Welch Two-Sample t-Test

Test Statistic FS: Welch t-test

$$t = \frac{\bar{X}_1 - \bar{X}_2}{\sqrt{\dfrac{S_1^2}{n_1} + \dfrac{S_2^2}{n_2}}}$$

Satterthwaite degrees of freedom:

$$\nu = \frac{\left(\dfrac{S_1^2}{n_1} + \dfrac{S_2^2}{n_2}\right)^2}{\dfrac{(S_1^2/n_1)^2}{n_1 - 1} + \dfrac{(S_2^2/n_2)^2}{n_2 - 1}}$$

Round $\nu$ down to the nearest integer. Compare $|t|$ to $t_{\nu, 1-\alpha/2}$ for a two-sided test.

D.5 Wilcoxon Rank-Sum Test

Key Facts

Null hypothesis: The two samples come from the same distribution.

When shape and spread are the same: the test becomes a test of location (median), not necessarily the mean.

Procedure: Rank all observations together, sum the ranks for each sample, compare to critical values or use the normal approximation.

D.6 Conceptual Points

Statistical Significance vs. Practical Significance

A very small p-value (e.g., $p < 0.001$) means the observed difference is statistically significant — unlikely to be due to chance alone. It does NOT mean the difference is large or practically important.

With large sample sizes, even tiny, meaningless differences become statistically significant. Always consider the effect size alongside the p-value.

Type I vs. Type II Errors

$H_0$ True$H_0$ False
Reject $H_0$Type I error ($\alpha$)Correct (Power = $1-\beta$)
Fail to reject $H_0$CorrectType II error ($\beta$)

Increasing df (more data) increases power — the ability to detect real differences.

E. MATLAB Monte Carlo Analysis

Why This Matters

The exam gives you MATLAB code and asks what integral it estimates. You need to systematically decode the code into a mathematical integral. Getting the constant factor right is the key challenge.

E.1 How Monte Carlo Integration Works

The MC Estimation Identity FS: Monte Carlo

If $X \sim \text{Uniform}(a_1, b_1)$ and $Y \sim \text{Uniform}(a_2, b_2)$, then:

$$E[g(X, Y)] = \frac{1}{(b_1-a_1)(b_2-a_2)}\int_{a_1}^{b_1}\int_{a_2}^{b_2} g(x,y)\,dy\,dx$$

Therefore: mean(H) where H = g(X,Y) estimates $E[g(X,Y)]$, which equals the integral of $g$ divided by the area.

Rearranging: The integral equals $\text{Area} \times E[g(X,Y)] = \text{Area} \times \texttt{mean(H)}$.

E.2 Reading MATLAB Code

MATLAB Translation Guide

MATLAB CodeMathematical Meaning
rand(1)*diff(rangeX)+rangeX(1)$X \sim \text{Uniform}(a_1, b_1)$
diff(rangeX)$b_1 - a_1$ (width of $X$ range)
log(x)$\ln(x)$ (natural logarithm)
log10(x)$\log_{10}(x)$
exp(X)$e^X$
E=mean(H)$E$ estimates $E[H(X,Y)]$

MATLAB Gotcha: log = $\ln$

In MATLAB, log(x) is the natural logarithm $\ln(x)$, NOT $\log_{10}(x)$. This trips people up constantly. Use log10 for base-10.

E.3 Systematic Decoding Procedure

How to Decode MC Code into an Integral

Identify the sampling region
Read rangeX and rangeY to find bounds $[a_1, b_1] \times [a_2, b_2]$.
Area $= (b_1 - a_1)(b_2 - a_2)$
Write down the code expression for H
Translate the MATLAB expression into math. Remember: log = $\ln$.
Apply the MC identity
mean(H) $= E[H] = \dfrac{1}{\text{Area}}\displaystyle\int\int H(x,y)\,dy\,dx$. So the integral being estimated is: $\text{Area} \times$ mean(H). But since mean(H) = E, check if the code already includes the area factor inside H.
If H already has a constant multiplier, simplify carefully.
Simplify to identify the integrand
$E[\text{H}] = \frac{1}{\text{Area}}\int\int H(x,y)\,dy\,dx$. So mean(H) estimates $\frac{1}{\text{Area}}\int\int H(x,y)\,dy\,dx$.

Worked Example E.1: Decode the exam MATLAB code

end
E=mean(H)

Step 1: Identify sampling region and area

$X \sim \text{Uniform}(2, 5)$,   $Y \sim \text{Uniform}(-1, 3)$.

Area $= (5-2)(3-(-1)) = 3 \times 4 = 12$.

Step 2: Translate the H expression

MATLAB: 24*exp(X)*sin(Y)/log(X^2)

In math: $H = \dfrac{24 \, e^X \sin(Y)}{\ln(X^2)}$

Now simplify: $\ln(X^2) = 2\ln(X)$, so:

$$H = \frac{24\,e^X \sin(Y)}{2\ln(X)} = \frac{12\,e^X \sin(Y)}{\ln(X)}$$

Step 3: Apply the MC identity

mean(H) estimates:

$$E[H] = \frac{1}{\text{Area}}\int_2^5\int_{-1}^3 H(x,y)\,dy\,dx = \frac{1}{12}\int_2^5\int_{-1}^3 \frac{12\,e^x\sin y}{\ln x}\,dy\,dx$$

Simplify:

$$E[H] = \frac{1}{12}\cdot 12 \int_2^5\int_{-1}^3 \frac{e^x \sin y}{\ln x}\,dy\,dx = \int_2^5\int_{-1}^3 \frac{e^x \sin y}{\ln x}\,dy\,dx$$

Result

E = mean(H) estimates:

$$I = \int_2^5\int_{-1}^3 \frac{e^x \sin y}{\ln x}\,dy\,dx$$

Note: there is no factor of 2 in the numerator. If an exam question claims the integral has $\frac{2e^x \sin y}{\ln x}$, that statement is false — it is off by a factor of 2.

Verification of the constant

Code constant = 24. After simplifying $\ln(X^2)$ to $2\ln X$, code constant becomes $24/2 = 12$. Dividing by Area = 12 gives $12/12 = 1$. So the integrand multiplier is just 1, confirming no extra constant.

F. Output Analysis

Why This Matters

Simulation output is random. Confidence intervals quantify uncertainty. The exam tests percentile bootstrap, empirical bootstrap, batch means, and t-CIs. Getting the rank calculations and flipped subscripts right is critical.

F.1 Percentile Bootstrap Confidence Interval

Percentile Bootstrap Procedure FS: Bootstrap CI

  1. Generate $N$ bootstrap replicate means $\bar{Y}_1^*, \bar{Y}_2^*, \ldots, \bar{Y}_N^*$.
  2. Sort them in ascending order: $\bar{Y}_{(1)}^* \leq \bar{Y}_{(2)}^* \leq \cdots \leq \bar{Y}_{(N)}^*$.
  3. For a $(1-\alpha)$ CI, compute the ranks:

    $j_{\text{low}} = N \cdot (\alpha/2) + C$

    $j_{\text{high}} = N \cdot (1-\alpha/2) + C$

    where $C = 1/2$ in this course.

  4. If $j$ is not an integer, interpolate linearly between adjacent order statistics.
  5. CI $= \left[\bar{Y}_{(j_{\text{low}})}^*,\; \bar{Y}_{(j_{\text{high}})}^*\right]$.

F.2 Empirical Bootstrap Confidence Interval

Empirical Bootstrap — Note the Flipped Subscripts! FS: Empirical Bootstrap

Compute the bootstrap deltas: $\delta_i^* = \bar{Y}_i^* - \bar{X}(n)$ for each bootstrap replicate.

Sort the deltas. The $(1-\alpha)$ CI is:

$$\left[\bar{X}(n) - \delta^*_{(1-\alpha/2)},\; \bar{X}(n) - \delta^*_{(\alpha/2)}\right]$$

The subscripts are flipped compared to what you might expect: the upper quantile of $\delta^*$ gives the lower bound of the CI, and vice versa.

Common Mistake: Subscript Flip

Students often write $[\bar{X} - \delta^*_{(\alpha/2)},\; \bar{X} - \delta^*_{(1-\alpha/2)}]$ (wrong order). Remember: the minus sign reverses the direction. The upper delta is subtracted to give the lower CI bound.

F.3 Batch Means Method

Batch Means

For a single long simulation run of $N$ observations:

  1. Discard initial transient (warm-up period).
  2. Divide remaining $n$ observations into $b$ batches of size $m$ ($n = bm$).
  3. Compute the mean of each batch: $\bar{Y}_j$ for $j = 1, \ldots, b$.
  4. Treat the $b$ batch means as approximately i.i.d. and apply a t-confidence interval.

Batches must be large enough that batch means are approximately independent.

F.4 t-Confidence Intervals

Standard t-CI

$$\bar{X}(n) \pm t_{n-1,\,1-\alpha/2} \cdot \frac{S(n)}{\sqrt{n}}$$

where $S(n)$ is the sample standard deviation and $t_{n-1,\,1-\alpha/2}$ is the critical value from the t-distribution with $n-1$ degrees of freedom.

F.5 Worked Example: Percentile Bootstrap CI

Worked Example F.1: 90% Bootstrap CI with $C = 1/2$

Data: 10 bootstrap replicate means:

$\bar{Y}^* = \{6.52,\; 7.01,\; 6.38,\; 7.24,\; 6.89,\; 6.45,\; 7.15,\; 6.73,\; 6.61,\; 6.98\}$

Step 1: Sort the data

$\bar{Y}_{(1)}^* = 6.38,\; \bar{Y}_{(2)}^* = 6.45,\; \bar{Y}_{(3)}^* = 6.52,\; \bar{Y}_{(4)}^* = 6.61,\; \bar{Y}_{(5)}^* = 6.73,$

$\bar{Y}_{(6)}^* = 6.89,\; \bar{Y}_{(7)}^* = 6.98,\; \bar{Y}_{(8)}^* = 7.01,\; \bar{Y}_{(9)}^* = 7.15,\; \bar{Y}_{(10)}^* = 7.24$

Step 2: Compute ranks for a 90% CI ($\alpha = 0.10$)

$j_{\text{low}} = N \cdot (\alpha/2) + C = 10 \times 0.05 + 0.5 = 0.5 + 0.5 = 1.0$

$j_{\text{high}} = N \cdot (1-\alpha/2) + C = 10 \times 0.95 + 0.5 = 9.5 + 0.5 = 10.0$

Step 3: Read off the order statistics

$j_{\text{low}} = 1.0$ → $\bar{Y}_{(1)}^* = 6.38$

$j_{\text{high}} = 10.0$ → $\bar{Y}_{(10)}^* = 7.24$

90% Percentile Bootstrap CI: $[6.38,\; 7.24]$

Note: If $j$ were non-integer, say $j = 2.3$, you would interpolate: $\bar{Y}^*_{(2)} + 0.3(\bar{Y}^*_{(3)} - \bar{Y}^*_{(2)})$.

G. Comparing Systems — Dudewicz & Dalal

Why This Matters

The Dudewicz & Dalal (D&D) two-stage procedure lets you select the best of $k$ system configurations with a guaranteed probability of correct selection $P^*$. This is a staple exam question.

G.1 Two-Stage Procedure Overview

Dudewicz & Dalal Procedure FS: D&D Table

Stage 1: Pilot run of $n_0$ observations per system
Run each of the $k$ systems for $n_0$ observations. Compute sample variances $S_i^2(n_0)$ for each system $i$.
Look up $h_1$ from the D&D table
Use the D&D table with parameters $P^*$, $k$, and $n_0$ to find the constant $h_1$.
$h_1$ depends on $P^*$, $k$, and $n_0$. This value comes directly from the formula sheet.
Compute required sample sizes $N_i$
For each system $i$:
$N_i = \max\!\left\{n_0 + 1,\; \left\lceil \dfrac{h_1^2\, S_i^2(n_0)}{(d^*)^2} \right\rceil\right\}$
Stage 2: Run additional observations
For each system $i$, run $N_i - n_0$ additional observations (beyond the pilot run).
“How many more runs?” = $N_i - n_0$, NOT $N_i$.
Compute weighted means and select the best
Compute weights $W_{i1}$ using $h_2$ from the D&D table (with $N_i - 1$ df). The weighted mean accounts for the two-stage sampling. Select the system with the best (largest or smallest) weighted mean.

Weight Formula

$$W_{i1} = \frac{d^*\, h_1\, n_0}{h_2(N_i - 1)\, N_i\, S_i(n_0)}$$

where $h_2$ is read from the D&D table for $N_i - 1$ degrees of freedom.

G.2 Bonferroni for $k > 2$ Systems

Bonferroni Inequality

When comparing $k > 2$ systems pairwise, there are $\binom{k}{2} = \frac{k(k-1)}{2}$ pairs.

To maintain overall confidence level $1-\alpha$, use significance level $\alpha / \binom{k}{2}$ for each pairwise comparison.

G.3 Worked Example

Worked Example G.1: D&D with two configurations

Setup: $k = 2$ systems, $n_0 = 20$ pilot observations each, $P^* = 0.95$, $d^* = 2.0$.

Pilot results: $S_1^2(20) = 244.3$, $S_2^2(20) = 67.3$.

From the D&D table ($P^* = 0.95$, $k = 2$, $n_0 = 20$): $h_1 = 2.453$.

Step 1: Compute $N_1$

$$N_1 = \max\!\left\{21,\; \left\lceil\frac{(2.453)^2 \times 244.3}{(2.0)^2}\right\rceil\right\} = \max\!\left\{21,\; \left\lceil\frac{6.017 \times 244.3}{4.0}\right\rceil\right\}$$

$$= \max\!\left\{21,\; \left\lceil\frac{1470.0}{4.0}\right\rceil\right\} = \max\{21,\; \lceil 367.5 \rceil\} = \max\{21,\; 368\} = 368$$

Step 2: Compute $N_2$

$$N_2 = \max\!\left\{21,\; \left\lceil\frac{6.017 \times 67.3}{4.0}\right\rceil\right\} = \max\!\left\{21,\; \left\lceil\frac{404.9}{4.0}\right\rceil\right\} = \max\{21,\; \lceil 101.2 \rceil\} = \max\{21,\; 102\} = 102$$

Step 3: Additional runs needed

Config A: $N_1 - n_0 = 368 - 20 = \mathbf{348}$ more runs.

Config B: $N_2 - n_0 = 102 - 20 = \mathbf{82}$ more runs.

Result

Configuration A (higher variance) needs 348 additional runs.

Configuration B (lower variance) needs 82 additional runs.

Higher variance systems require dramatically more observations to achieve the same precision.

H. Variance Reduction

Why This Matters

Variance reduction techniques let you get tighter confidence intervals from the same number of simulation runs. The exam tests whether you can compute covariances analytically and determine when each technique helps or hurts.

H.1 Common Random Numbers (CRN)

CRN Variance Formula FS: Variance Reduction

When comparing systems 1 and 2 using the same random numbers:

$$\text{Var}[\bar{Z}(n)] = \frac{\text{Var}(X_{1j}) + \text{Var}(X_{2j}) - 2\,\text{Cov}(X_{1j}, X_{2j})}{n}$$

where $Z_j = X_{1j} - X_{2j}$.

CRN helps when $\text{Cov}(X_1, X_2) > 0$ (positive correlation reduces the variance).

CRN hurts when $\text{Cov}(X_1, X_2) < 0$ (negative correlation increases variance).

H.2 Antithetic Variates

Antithetic Variates Variance Formula

Pair each run using $U$ with a run using $1-U$. Let $X^{(1)} = g(U)$ and $X^{(2)} = g(1-U)$, and average them:

$$\text{Var}\!\left[\frac{X^{(1)} + X^{(2)}}{2}\right] = \frac{\text{Var}(X^{(1)}) + \text{Var}(X^{(2)}) + 2\,\text{Cov}(X^{(1)}, X^{(2)})}{4}$$

For $n$ pairs:

$$\text{Var}[\bar{X}(n)] = \frac{\text{Var}(X^{(1)}) + \text{Var}(X^{(2)}) + 2\,\text{Cov}(X^{(1)}, X^{(2)})}{4n}$$

Antithetic helps when $\text{Cov}(X^{(1)}, X^{(2)}) < 0$ (negative correlation). This is why we use $1-U$: if $g$ is monotone, $g(U)$ and $g(1-U)$ tend to be negatively correlated.

H.3 Control Variates

Control Variates FS: Control Variates

Use a correlated variable $Y$ with known mean $E[Y]$ to reduce variance of estimator for $E[X]$:

$$X_c = X + a(Y - E[Y])$$

Optimal coefficient:

$$a^* = -\frac{\text{Cov}(X, Y)}{\text{Var}(Y)}$$

Note the minus sign. The variance reduction achieved is proportional to $\rho^2_{XY}$ (the squared correlation).

H.4 Key Identity for Uniform Moments

Uniform Moments

If $U \sim \text{Uniform}(0,1)$:

$$E[U^k] = \frac{1}{k+1}$$

So: $E[U] = 1/2$,   $E[U^2] = 1/3$,   $E[U^3] = 1/4$,   $E[U^4] = 1/5$, etc.

H.5 Analytical Covariance Computation

Covariance Formula

$$\text{Cov}(X, Y) = E[XY] - E[X]\,E[Y]$$

Expand $XY$ as a polynomial in $U$, then use $E[U^k] = 1/(k+1)$.

Worked Example H.1: Compute $\text{Cov}(X_1, X_2)$ where $X_1 = (U-2)^2$, $X_2 = U^2$

Given: $U \sim \text{Uniform}(0,1)$, $X_1 = (U-2)^2$, $X_2 = U^2$.

Step 1: Compute the uniform moments we need

$E[U] = 1/2$,   $E[U^2] = 1/3$,   $E[U^3] = 1/4$,   $E[U^4] = 1/5$

Step 2: Compute $E[X_1]$

$X_1 = (U-2)^2 = U^2 - 4U + 4$

$E[X_1] = E[U^2] - 4E[U] + 4 = \frac{1}{3} - 4\cdot\frac{1}{2} + 4 = \frac{1}{3} - 2 + 4 = \frac{1}{3} + 2 = \frac{7}{3}$

Step 3: Compute $E[X_2]$

$E[X_2] = E[U^2] = \frac{1}{3}$

Step 4: Compute $E[X_1 X_2]$

$X_1 X_2 = (U-2)^2 \cdot U^2 = (U^2 - 4U + 4) \cdot U^2 = U^4 - 4U^3 + 4U^2$

$E[X_1 X_2] = E[U^4] - 4E[U^3] + 4E[U^2] = \frac{1}{5} - 4\cdot\frac{1}{4} + 4\cdot\frac{1}{3} = \frac{1}{5} - 1 + \frac{4}{3}$

Common denominator (15): $= \frac{3}{15} - \frac{15}{15} + \frac{20}{15} = \frac{8}{15}$

Step 5: Compute the covariance

$\text{Cov}(X_1, X_2) = E[X_1 X_2] - E[X_1]\,E[X_2] = \frac{8}{15} - \frac{7}{3}\cdot\frac{1}{3} = \frac{8}{15} - \frac{7}{9}$

Common denominator (45): $= \frac{24}{45} - \frac{35}{45} = -\frac{11}{45}$

Result and Interpretation

$\text{Cov}(X_1, X_2) = -\dfrac{11}{45} \approx -0.244$

Since $\text{Cov} < 0$:

  • CRN would INCREASE variance (the $-2\text{Cov}$ term becomes $+2 \times 11/45$, making things worse).
  • Antithetic variates would benefit (the $+2\text{Cov}$ term is negative, reducing variance).

Common Mistakes with Variance Reduction

  • Sign confusion in CRN: CRN has $-2\text{Cov}$ in the variance formula. Positive Cov reduces variance; negative Cov increases it.
  • Sign confusion in Antithetic: Antithetic has $+2\text{Cov}$ in the variance formula. Negative Cov reduces variance; positive Cov increases it.
  • Forgetting the minus in control variates: $a^* = -\text{Cov}(X,Y)/\text{Var}(Y)$. The minus sign is part of the formula.
  • Arithmetic errors with fractions: When computing $E[U^k]$, always convert to a common denominator before adding/subtracting.
