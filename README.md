# SSA Exam Prep — iLearn

Static study site for the **Stochastic Simulation & Analysis** exam. All pages are self-contained HTML with MathJax for LaTeX rendering — just open in a browser.

## Pages

| File | Content |
|------|---------|
| `index.html` | Home — course overview, exam format, formula reference, study tips |
| `page1.html` | Foundations & Probability |
| `page2.html` | Statistical Methods & Advanced Analysis |
| `page3.html` | Resit Preparation & Theory Deep-Dives |
| `exam.html` | Reconstructed Resit Exam |

## Usage

Open `index.html` in any browser. No build step or server required.
